## 목표
현재 완료 탭은 주별로 분리해서 12주차, 13주차 이렇게 표시되고 있음
## 구현 항목
- [x] 일별, 반주, 주별, 월별 필터 구현
  - 서버: 태스크 완료 시 completed_at 프론트매터 자동 추가, 기존 태스크는 mtime 보정
  - 프론트: 완료 컬럼 상단에 [일별] [반주] [주별] [월별] 필터 칩
  - 반주 = 월~수(전반) / 목~일(후반), 날짜 범위 레이블 표시

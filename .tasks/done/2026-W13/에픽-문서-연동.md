## 목표
설정에서 에픽을 생성하면 docs/epics/ 에 해당 에픽 문서가 자동 생성되고, 에픽 진행률 바에서 에픽 문서로 바로 이동할 수 있게 연동.

## 구현 항목
- [x] 에픽 생성/저장 시 docs/epics/{epic-id}.md 문서 자동 생성 (없으면)
  - PUT /api/config에서 epics 배열 순회, fs.access로 존재 확인 후 없으면 생성
  - 템플릿: 왜/결정된 것/미결 사항/태스크 목록/메모
- [x] 에픽 진행률 바에서 에픽명 클릭 시 해당 문서로 이동
  - onViewDoc 콜백으로 문서 탭 전환 + docs/epics/{id}.md fetch
  - 에픽명 hover 시 underline + 파란색 힌트
- [x] 에픽 삭제 시 문서는 유지 (orphan으로 남김)
  - 서버에서 삭제 로직 없음, 문서는 그대로 남음

## 참고
- 기존 docs/epics/ 폴더는 DOC_TYPES에 이미 정의됨
- 에픽 문서 템플릿: 왜/결정된 것/미결 사항/태스크 목록/메모

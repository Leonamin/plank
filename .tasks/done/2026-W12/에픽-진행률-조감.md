## 목표
보드 상단에 라벨별 태스크 진행률을 프로그레스 바로 요약 표시. 라벨 = 에픽 개념으로 활용.

## 구현 항목
- [x] 라벨별 태스크 수 집계 (전체 / done 컬럼) — EpicBar stats 계산
- [x] 보드 상단에 라벨별 프로그레스 바 렌더링 (예: `dev 6/10 60%`) — epic-chip-bar + count
- [x] 프로그레스 바 색상은 라벨 color 사용 — epic-chip-fill에 라벨 color 적용
- [x] 태스크가 없는 라벨은 표시하지 않음 — filter(s => s.total > 0)
- [x] 라벨 필터 토글과 연동 (칩 클릭 시 필터 겸용) — onToggleFilter로 통합

## 메모
- 기존 .progress-bar, .progress-fill CSS 재활용
- 라벨로 부족하면 나중에 별도 epic 필드 확장 가능
- 별도 브랜치에서 작업

# 목표
칸반 목록의 크기를 브라우저 창의 높이로 제한하고 취소 영역, 보류 영역은 항상 보이게 해줘

- [x] 칸반 목록 자체는 브라우저 창의 높이, 그리고 목록 개별로 스크롤바 배치
- [x] 스크롤바는 모던 디자인으로 기본 브라우저 디자인의 스크롤바로 하지마
- [x] 완료 칸반의 취소 영역, 보류 영역을 별개의 행으로 배치 같은 행에 나눠서 배치하지 말아줘

## 목표
보드 탭 상단에 라벨 칩을 나열하고, 클릭하면 해당 라벨이 붙은 태스크만 표시되도록 필터링 기능 추가.

## 구현 항목
- [x] 보드 상단에 라벨 칩 목록 렌더링 (config.labels 기반) — EpicBar 컴포넌트
- [x] 칩 클릭 시 활성/비활성 토글 (복수 선택 가능) — activeFilters state
- [x] 활성 필터가 있으면 해당 라벨 포함 태스크만 컬럼에 표시 — filtered 변수로 Column에 전달
- [x] 필터 초기화 버튼 ("전체 보기") — epic-chip-clear
- [x] 필터 활성 시 컬럼 카운트도 필터된 수로 업데이트 — filtered 태스크 전달로 자동 반영

## 메모
- 기존 picker-chip CSS 클래스 재활용 가능
- 별도 브랜치에서 작업 (feature/label-filter)
